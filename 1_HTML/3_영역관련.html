시맨틱(semantic)태그

컴퓨터는 단순 문장이나 문장 위치, 모양만으로 내용을 이해하기 힘드므로 컴퓨터가 각 문장에 대한 의미를 이해할 수 있도록 요소를 사용하여 명확하게 나타내기 위함이다.
이전에는 div에 id를 집어넣어서 각 div를 구분했어도 실질적인 의미를 갖진 못했음
HTML5로 넘어오면서 div이면서 거기에 의미까지 포함된 시맨틱 태그가 등장.

body안에 사용하며 여러 번 사용 가능, 웹 문서에서 머리말을 나타날때 사용함
제목, 제목 설명, 문장 등을 나타내는 요소를 포함한다
navigation, 웹 문서에서 다른 웹 문서나 문서 내 다른 부분으로 이동하는 부분을 나타내고자 할 때 사용
웹 문서의 광고나 링크 모음 같은 부가적인 정보를 구분하기 위해 사용
웹 문서에서 주요 내용을 가진 본문을 나타낼 때 사용 실제로 나타내고자 하는 내용을 가지고 있으므로 기계에 의해 중요한 부분으로 인식되어 처리 여러 개의 article 허용
웹 문서의 저자 정보, 저작권 정보, 이용 조건 등을 나타내기 위해 사용
