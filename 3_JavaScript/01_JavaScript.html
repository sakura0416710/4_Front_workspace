자바 스크립트 개요

스크립트 언어

기본 프로그램의 동작을 사용자의 요구에 맞게 수행되도록 해주는 용도로 사용
매우 빠르게 배우고 짧은 소스코드 파일로 상호작용하도록 고안됐다.

자바스크립트

웹 브라우저에서 많이 사용하는 인터프리터 방식의 객체지향 프로그래밍 언어

자바스크립트 사용법

1.태그에 직접 간단한 소스코드를 작성해서 실행하는 방법 : inline방식
2. script태그에 작성하여 실행하는 방법 : internal방식 (내부방식)
3. 별도의 js파일로 작성해서 가져다 사용하는 방법 : external방식 (외부방식)

응용해보기

2. 태그에 직접 자바스크립트 코드를 작성하여 실행 : 인라인inline 방식

2. script태그 안에 함수 단위로 소스 작성하여 실행 : internal 방식( 내부방식 )

3. 별도의js파일로 작성하여 실행 : external (외부방식)

데이터 입력

window.confirm()

어떤 질문에 대해 예/아니오의 결과를 얻을 때 사용.
확인 버튼과 취소 버튼이 나타나며 확인 버튼 클릭 시 true 반환, 취소 버튼 클릭 시 false 반환

window.prompt()

텍스트필드와 확인, 취소 버튼이 있는 대화상자
출력하고 입력한 메세지 내용을 리턴값으로 돌려받음
메세지를 입력한 후 확인 버튼을 누르면 입력한 메세지, 취소버튼 누르면 null리턴
